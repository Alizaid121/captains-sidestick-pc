# Captain's Sidestick — PC Server

PC server application for Captain's Sidestick flight simulator controller.

## Features
- Virtual Xbox 360 controller via ViGEmBus
- WebSocket server on port 8888
- QR code for easy phone connection
- Live axis and button display
- Works with X-Plane 12, MSFS, DCS and all simulators

## Requirements
- Windows 10/11 64-bit
- Run as Administrator

## Installation
1. Download CaptainsSidestick-Setup-x.x.x.exe from Releases
2. Run as Administrator
3. Follow installer — ViGEmBus installs automatically
4. Launch from desktop shortcut
